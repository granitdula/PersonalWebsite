Projects

Reward Rush!

Reward Rush is an augmented reality point scoring mobile game by finding objects around the hospital, created for Great Ormond Street Hospital as a University project. Its aim is to help make the experience for children in the hospital less daunting. This project was liked so much by our clients that we were offered money to continue it.

VR Sports Trainer

This is a virtual reality sports trainer application I built with two other team members for a University project for our clients Immersive Ltd. It gamifies the scenario of training boxing by having the user score points by hitting and blocking incoming targets within a certain time limit. It uses the Microsoft Mixed reality and Microsoft Kinect to read human gestures and diplay the virtual world.

Bio-Reactor

A team of 10 including me created this Bio-Reactor Controller that controls the temperature, pH and stirring speed of a reactor for a facility in Uganda making Tuberculosis vaccines. This was part of an engineering module in our university course. It required using a MSP430 Microcontroller to program the control systems and send it the data to the UI shown above and allow users to control the values.
